web性能优化（三）：SEO

web性能 SEO 搜索引擎优化

什么是SEO

SEO，Search Engine Optimization，中文意译为“搜索引擎优化”，指在了解搜索引擎自然排名机制的基础上，对网站进行内部及外部的调整优化，改进网站在搜索引擎中关键词的自然排名，获得更多的展现量，吸引更多目标客户点击访问网站，从而达到网络营销及品牌建设的目标。

站长应该将优化的重心和出发点主要放在用户体验上,因为用户才是网站内容的受众，是他们通过搜索引擎找到了这个网站。过度专注于用特定的技巧获取搜索引擎自然搜索结果的排名不一定能够达到期望的结果。讲人话，SEO就是让网站以最理想的姿态出现在搜索引擎的结果中，但是您的最终的服务对象是您的用户而不是搜索引擎。

没有经过搜索引擎优化的网站有以下特征：

  1. 网页中大量采用图片或者Flash等富媒体（Rich Media）形式，没有可以检索的文本信息，而SEO最基本的就是文章SEO和图片SEO；
  2. 网页没有标题，或者标题中没有包含有效的关键词；
  3. 网页正文中有效关键词比较少（最好自然而重点分布，不需要特别的堆砌关键词）；
  4. 网站导航系统让搜索引擎“看不懂”；
  5. 大量动态网页影响搜索引擎检索；
  6. 没有其他被搜索引擎已经收录的网站提供的链接；
  7. 网站中充斥大量欺骗搜索引擎的垃圾信息，如“桥页（也叫门页，过渡页）”、颜色与背景色相同的文字；
  8. 网站中缺少原创的内容，完全照搬硬抄别人的内容等。

如何进行SEO

1. SEO基础

1.1 创建独特、准确的页面标题
我们都造呢，html文件的<head>标签中可以有<title>标签，一个页面的标题概括了这个页面的主题和主要内容，title中的内容不仅会紧跟着该页面的icon显示，而且会显示在浏览器的搜索结果中。我们应该为每个页面创建一个简短、具有概括性的标题。
1.2 恰当地使用描述元标签
某些浏览器可能会使用描述元标签来生成网页的摘要。

2. 优化网站结构

2.1 优化url的结构
使用简单易懂的url表达网页信息，这有助于:

  • 站长自己组织管理网站
  • 搜索引擎有效地抓取文件
  • 相比于url中的ID，网站浏览者会对文字更感兴趣，感兴趣程度也决定他们是否点击该链接

正确使用url：

  • 使用文字
  • 使用简单的目录结构，避免多层嵌套的子目录
  • 为同一文档提供统一的url，若有多个url，可用重定向到所期望的url或使用首选域URL或rel=“canonical”的链接元素。

2.2 让自己的网站更易于检索和浏览
网站的导航功能对于搜索引擎而言是非常重要的，虽然Google的搜索结果都是指向每一个特定的页面，但Google同时也会智能地去理解这一页到底在整个网站结构中处于什么地位。
正确使用导航功能：

  • 基于主页来规划网站的导航功能，整个网页的文件相当于一个文件系统，主页相当于一个根目录，所有页面都可以从主页出发找到
  • 通过“面包屑导航”给用户带来更多便捷。面包屑导航可以让访客清晰地看到当前页面所处网站位置，目录结构清晰明了，方便访客访问网站任意位置
  • 预先考虑到用户可能截取部分URL的情况，访客可能凭借自己的主观推测某些url可能显现的网页，尽可能考虑这些情况避免出现404
  • 导航中尽量使用文字链接
  • 创建自然的层叠结构，不要过于细分网站结构而创建复杂的导航链接
  • 可以创建有意义的404网页
  • 准备好两份网站地图 : 针对用户的以及针对搜索引擎的。给用户的应该是以层级式列表的方式而给浏览器的是以XML网站地图文档方式。

3. 优化内容

我之前有些过关于网站内容的优化，这里作一点补充。

3.1 网站提供高质量的内容和服务
毋庸置疑，高质量的内容和服务会吸引大量的访客，得到认可的网站会提高在用户和搜索引擎中的声望，为网站带来更多的流量，造成良性循环。

在撰写网站内容时，预先考虑到一种事物的多种名称叫法，通过综合使用各种具有潜在搜索可能性的关键词，兼顾到不同用户的搜索习惯，那么将会收到更好的效果。
提供高质量的内容和服务的参考方法：

  • 撰写容易阅读的文字
  • 围绕内容有条理地组织内容
  • 提供新颖、原创的内容
  • 为用户负责，不要欺骗浏览器

3.2 写好锚文本
合适的锚文本会使用户明白文本链接的内容，更倾向于点击链接。

3.3 优化图片的使用

  • 学会使用alt属性，使得浏览器由于某些原因无法显示图片时，让用户对此处显示的图片有一个大致的了解
  • 将图片保存在一个特定的目录下，便于管理
  • 使用常见的图片格式

4. 处理页面的抓取

4.1 防止搜索引擎抓取某些页面
robots.txt文件主要用来告知搜索引擎它们是否有权抓取您网站的特定部分。该文件一定要严格的命名为“robots.txt”，并被放置在网站的根目录下。

  • 生成robots.txt文件
  • 给robots元标签加上NOINDEX标识
  • 使用.htaccess文件对需要保护的目录加密
  • 使用网站站长工具将已经被抓取的网页从搜索结果中移除

4.2 使用rel=“nofollow”
如果把某一链接的“rel”属性设置成“nofollow”表示不希望Google继续跟踪抓取这些特定链接所指向的页面。
应用场景：

  • 垃圾留言。比如，您网站上的某条留言中有个链接，您并不明白该链接指向什么内容，如果是指向某些垃圾信息，您的网站声誉将会受到影响。如果使用了nofollow属性，则不会损坏您网站的声誉。
  • 引用链接。您在某个地方引用了一个链接，那么您会将自己的pr传递给链接所指向的页面。如果您需要引用一个垃圾页面，使用nofollow属性则不会将您网站的声誉值传递给引用的页面。

5. 用正确的手段推广网站

5.1 通过增加反向链接
链向您网站的链接需要逐渐积累，因为人们总是通过搜索或者其他途径认识到了您的网站，之后才会链接向它。
5.2 学会使用社交网络
5.3 充分利用网站站长工具
利用网站站长工具可以让Googlebot的抓取更加顺利。您还可以获取分析服务的数据包，对访问者如何与您的网站进行互动得出更详尽的分析，获取访问者主要是通过哪些关键词访问到自己的网站。
