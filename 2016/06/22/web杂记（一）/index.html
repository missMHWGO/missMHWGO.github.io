web杂记(一)

web window
为什么要在函数定义时将window作为参数传递到函数中去:

  • javascript的scope chain作用域链，在当前作用域中如果没有该属性（局部变量）则向上一层作用域中寻找，一直到最上层，也就是window。也就是说全局变量和下级作用域都是window的一个属性。
  • 通过传入window变量，使得window由全局变量变为局部变量，访问window时，不需要将作用域链回退到顶层作用域，这样可以更快的访问window，提高程序效率。
  • 将 window 作为参数传入，可以在压缩代码时进行优化。
